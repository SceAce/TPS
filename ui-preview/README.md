# TPS UI Preview

This folder is an isolated fake-data UI prototype for reviewing the next visual direction.

It is not connected to the Android app, backend, database, Hilt, or Retrofit. Do not copy this folder into `app/`.

Open `index.html` through the local preview server to review:

- Homepage visual system
- Pull-to-refresh spinner states
- Product card layout
- Bottom navigation and publish FAB
- Message, order, and profile style references
