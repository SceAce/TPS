# TPS 测试用例表

> 文档说明：本文件列出主要测试用例、输入条件与预期结果。


**版本：** v1.0  
**日期：** 2026-05-05  
**用途：** 作为课程“软件测试”阶段的详细测试用例记录

---

## 1. 测试环境

- 客户端：Android 模拟器 / 真机
- 服务端：Spring Boot 3.x
- 数据库：MySQL 8.x
- 鉴权：JWT

---
